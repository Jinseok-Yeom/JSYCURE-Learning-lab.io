Security Learning Datasets

이 페이지는 Suricata 및 보안 실습 환경에서 수집한 교육용 CSV 데이터셋을 정리한 공간입니다. 각 데이터셋은 보안 로그 분석, 라벨링, Feature Engineering, MLP 기반 IDS 실습을 목적으로 구성되어 있습니다.

Beginner Level

Suricata 로그 구조를 처음 접하는 학습자를 위한 단계입니다. CSV 형식, 기본 컬럼, 라벨 구조를 이해하는 데 초점을 둡니다.

Intermediate Level

Feature Engineering, 데이터 밸런싱, 학습용 데이터셋 구성 등 실제 MLP 실습으로 넘어가기 위한 중간 단계입니다.

Advanced Level

외부 로그 예측, threshold 최적화, ROC/PR Curve 기반 해석 등 연구 및 고급 실습 흐름으로 확장하기 위한 단계입니다.

How to Use These Datasets

  • 기초 단계에서는 Flow Sample과 Alert Sample로 CSV 구조와 라벨링 개념을 먼저 익힙니다.
  • 중간 단계에서는 Feature Engineering Sample과 Balanced Dataset으로 학습용 데이터셋 구성을 연습합니다.
  • 고급 단계에서는 External Prediction Input Sample과 Threshold Optimization Practice를 통해 실제 예측 및 평가 흐름을 실습합니다.
  • 각 데이터셋은 교육 목적의 샘플이므로 실제 운영 원본 로그 대신 비식별화 및 실습용 구조로 제공하는 것을 권장합니다.
