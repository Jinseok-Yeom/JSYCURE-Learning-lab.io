Learning Guides

Suricata 기반 교육 및 실습 흐름을 단계별로 정리한 가이드입니다.

1. Suricata 로그 수집

eve.json에서 flow 및 alert 이벤트를 수집하고, flow_id 기준으로 공격 여부를 매핑하는 과정을 다룹니다.

2. CSV 변환 및 라벨링

BENIGN=0, ATTACK=1 기준으로 라벨을 정의하고, CSV 형태의 학습용 데이터셋으로 정리하는 과정을 설명합니다.

3. Feature Engineering

total_pkts, total_bytes, duration, packet_rate, byte_rate, flow_ratio, pkt_ratio, byte_ratio와 같은 파생 feature 생성 과정을 설명합니다.

4. MLP 학습

PyTorch 기반 MLP 모델을 사용하여 GPU 환경에서 이진 분류 학습을 수행하는 흐름을 설명합니다.

5. Threshold 최적화

Precision 우선 기준으로 threshold를 선택하고, 조건을 만족하지 못할 경우 F1-score 기준으로 fallback하는 방식을 설명합니다.
